# fashion-ecommerce-analysis
Sales and customer behavior analysis for a fashion e-commerce store using Excel and Power BI dashboards.


## Objectives
- Analyze sales performance by city and time period
- Identify top-selling products and categories
- Understand customer purchasing patterns (order value, frequency)
- Provide insights to support marketing and inventory decisions

## Tools Used
- Excel (data cleaning, pivot tables, charts)
- Power BI (interactive dashboards and KPIs)
